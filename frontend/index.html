Phase 1 — Development
Spring Boot 3.3
Java 17 REST API. Auto-seeds 6 products. CORS enabled for frontend.
Phase 2 — Database
MySQL 8
Products and Orders tables auto-created by Hibernate on first run.
Phase 3 — Version Control
Git + GitHub
Push code to GitHub. Jenkins polls for changes and triggers pipeline automatically.
Phase 4 — Build
Apache Maven
Compiles Java code, downloads dependencies, packages as shopflow.jar.
Phase 5 — Testing
JUnit 5
6 tests: price validation, stock check, discount calc, GST, order total.
Phase 6 — Containerize
Docker
App packaged as Docker image using eclipse-temurin:17. Reproducible anywhere.
Phase 7 — Orchestration
Docker Compose
mysql + app services. App waits for MySQL healthcheck before starting.
Phase 8 — CI/CD
Jenkins
Clone → Build → Test → Docker Build → Auto-deploy. No manual steps.

🚀 Run Locally — Complete Steps

# 1. Build the JAR (run from project root)
mvn clean package -DskipTests

# 2. Start MySQL + App together
docker compose up --build

# 3. Backend API is live at:
#    http://localhost:9090

# 4. Open frontend (no server needed)
#    Double-click frontend/index.html in your browser

# 5. Stop everything
docker compose down

⚙️ Jenkins CI/CD Setup

# Start Jenkins container (run once)
docker run -d --name jenkins -p 8080:8080 -p 50000:50000 \
  -v jenkins_home:/var/jenkins_home \
  -v /var/run/docker.sock:/var/run/docker.sock \
  jenkins/jenkins:lts

# Give Jenkins permission to use Docker
docker exec -u 0 -it jenkins bash
chmod 666 /var/run/docker.sock && exit
docker restart jenkins

# Open Jenkins UI: http://localhost:8080
# Get password:
docker exec jenkins cat /var/jenkins_home/secrets/initialAdminPassword

# In Jenkins: New Item → Pipeline → paste Jenkinsfile → Build Now

☁️ Free Cloud Deployment
🟢 Render
⭐ Recommended — Free
Easiest free cloud for Spring Boot. Auto-deploys on every GitHub push.
  1. Go to render.com → Sign up free
  2. New → Web Service → Connect GitHub repo
  3. Build Command: mvn package -DskipTests
  4. Start Command: java -jar target/shopflow.jar
  5. Add a free PostgreSQL database
  6. Set env vars (DB URL, user, pass)
  7. Click Deploy → Live in 5 min ✅
🔵 Railway
$5 Free Credits
One-click Docker deploy from GitHub. Has native MySQL support.
  1. Go to railway.app → Login with GitHub
  2. New Project → Deploy from GitHub repo
  3. Add MySQL Plugin from marketplace
  4. Railway auto-detects Dockerfile
  5. Set SPRING_DATASOURCE_URL from MySQL plugin
  6. Deploy → Done ✅
🟠 Oracle Cloud
Always Free Forever
Free VM with 1GB RAM. Best for running Docker + Jenkins 24/7.
  1. Sign up at cloud.oracle.com
  2. Create Always Free Ubuntu VM (Ampere A1)
  3. SSH into VM: ssh ubuntu@YOUR_IP
  4. Install Docker: sudo apt install docker.io docker-compose -y
  5. Clone your GitHub repo on the VM
  6. mvn package -DskipTests && docker compose up -d
  7. Open port 9090 in Oracle firewall → Live ✅
🟣 Fly.io
Free Hobby Tier
Deploy Docker containers globally. Free for small apps with Postgres.
  1. Install: curl -L https://fly.io/install.sh | sh
  2. flyctl auth login
  3. flyctl launch (auto-detects Dockerfile)
  4. flyctl postgres create
  5. flyctl postgres attach --app shopflow
  6. flyctl deploy → Live globally ✅
